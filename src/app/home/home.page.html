About

Formana's purpose
With the frequent surveying and collection of data and information from individuals on campus, it is inevitable to create multiple forms having the same purpose. With this project, it aims to lessen the creation of multiple forms with the same agenda. Its target is to save the data collected in one single database that is accessible to all administrators unlike in other platforms where only the creator of the form is able to access the results. In this way, retrieval of information will be way easier and replication of survey results will be lessened.

This application is developed in order to make it more convenient for the students and administrators in collecting needed information. The application offers the feature to make customized form templates which will suit their objective. Along with this, pre-made templates will also be available which can serve as basis in form creation.

Formana will be implemented through a web application and mobile application. The mobile application will be accessible for the student users, this is where they will be able to preview and answer the forms they are targeted to respond to. On the other hand, the web application will contain a dashboard and will mostly be for administrator use.
